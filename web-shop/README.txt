Порядок установки сервера с базой и сайта:
1. Разархивировать папку web-shop;
2. Добавить в папку web-shop node-modules;
3. Удалить package-lock.json;
4. В консоли прописать путь до папки web-shop;
5. Прописать в консоли npm install;
6. После установки всех расширений и дополнений(создастся новая папка package-lock.json)
прописать в консоли json-server --watch db.json для запуска сервера базы данных;
7. Запустить локальный сервер командой npm run serve.
